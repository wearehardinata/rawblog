Kali ini saya mau melanjutkan tutorial sebelumnya nih, sebelumnya itu kita udah konfigurasi repo lokal kita dan repo di github/gitlab. Nah untuk kali ini saya mau nambahin dikit nih tutorial agar kita gak perlu di suruh masukin username & password sewaktu push, atau pull repo kita yang di GitHub/GitLab.

1. Kita clone repo kita yang udah di host di GitHub/GitLab 

    git clone https://github.com/user/repo.git Latihan\ Git

2. Selanjutnya kita generate ssh keys pc kita 

    ssh-keygen

3. Ketik perintah dibawah, copy ssh keys yang ditampilkan dan daftarkan pada GitHub/GitLab 

    cat /root/.ssh/id_rsa.pub

Sampai point ke- 3 ini harusnya tidak ada masalah, alias nanti ketika kita push atau pull dari dan ke GitHub kita ga perlu masukin username atau password lagi.
