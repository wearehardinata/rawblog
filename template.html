Git, push and commit

Langsung aja, buat yang belom punya akun (GitHub / GitLab) silahkan daftar dahulu. Santai aja, gampang kok daftarnya. Habis daftar jangan lupa kita buat repo baru, kasih nama repo -nya terserah kalian sob.

Selanjutnya kita siapkan juga direktori kerja kita, buat direktori Latihan Git, lalu masuk direktori tersebut. 

    mkdir Latihan\ Git
    cd Latihan\ Git
    code .

Pada direktori ini kita coba buat file baru, simpan dan beri nama index.php (atau nama apa aja terserah kamuu :D). Selanjutnya, kita jalankan perintah dibawah. Perintah ini akan membuat direktori kerja kita dikenal sebagai local git repositori. 

    git init

Lohh kok cuma segitu?? hehehe iya cukup ini aja si Git udah ngerti kok, gak kayak kamuu gak peka-peka :p. Perintah diatas akan mendaftarkan direktori kerja kita beserta file didalamnya kedalam repo local git kita. Lalu agar repo kita sync dengan yang udah kita buat di GitHub/GitLab tadi, kita jalankan perintah dibawah. 

    git remote add origin https://github.com/user/repo.git
